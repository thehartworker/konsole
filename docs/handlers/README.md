# Handler-Detail-Specs

Detail-Specs pro Backend-Handler (W1 bis W6) werden hier sukzessive abgelegt, sobald sie über die Kurz-Specs in `WORKFLOW_HANDLERS_v0.1.md` hinaus konkretisiert werden (Datenmodell, Prompts, Test-Referenzfälle).

Ein Verzeichnis pro Handler, benannt nach dem Handler-Slug, etwa `w1-pressemitteilung-drafter/`.
