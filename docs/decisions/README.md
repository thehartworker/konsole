# Design-Decisions

Jede substanzielle Konzept-Entscheidung wird hier als eigene Markdown-Datei dokumentiert, bevor der zugehörige Code gebaut wird (siehe `AGENTS.md` §3.2).

**Dateiname-Muster:** `YYYY-MM-DD_kurztitel.md`

**Status-Werte:** `vorgeschlagen` (Entwurf, wartet auf Review) → `akzeptiert` (freigegeben) → `überholt` (durch neuere Entscheidung ersetzt)
